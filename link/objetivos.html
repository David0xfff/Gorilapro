Objetivos

Desarrollar un prototipo de una tarjeta electrónica para poder probar aplicaciones de software en hardware, las cuales se podrán programar en assembler y/o lenguaje C++; con la posibilidad de conexión de periféricos como sensores y actuadores.


Aportar a materias como sistemas digitales II, sistemas digitales II, en trabajos extra clase.


Crear una plataforma web amigable de manera informativa, para que el usuario pueda conocer ejemplos y aplicaciones de desarrollo que se pueden realizar con el prototipo sistema de desarrollo.


Para el prototipo sistema de desarrollo se utilizaran los siguientes chips, sensores y actuadores:


  • Controlador MAX7219/7221
  • Sensor de temperatura LM35
  • Teclado matricial 4X4
  • Pantalla LCD 20X4
  • Módulo I2c85741
  • Relé 10A
  • Driver L293D
  • Módulo de comunicación bluetooth HC-05
  • Ocho Display 7 segmentos



    • Objetivo General



    Diseñar y construir un prototipo, basado en microcontroladores (Microchip pic18F4550) que permita desarrollo de equipos de una forma ágil, apoyados con sistemas modulares de sensores y actuadores.


    Desarrollar una página web donde el usuario pueda encontrar explicación teórica y práctica con videos tutoriales, donde se explicara la creación de software y poder descargar códigos de programación hechos en los tutoriales.



    Objetivos Especificos



  • Diseñar un diagrama electrónico, el cual muestra el conexionado de los componentes electrónicos.


  • Desarrollar módulos de software para probar el correcto funcionamiento del hardware del prototipo.


  • Diseñar el PCB del prototipo para el ensamblaje de los sistemas y/o módulos electrónicos con el microcontrolador.


  • Desarrollar página web, en la cual se compartirá el manual de consulta con la descripción de las etapas del desarrollo del prototipo, al igual que poseerá videos de los aspectos más importantes de este proceso y algunos ejemplos y aplicaciones.

Pantalla LCD 20x4



La pantalla de cristal líquido (LCD) proporciona una forma rentable de colocar una unidad de salida de texto para un microcontrolador. Como hemos visto en el tutorial anterior, los LED o los 7 segmentos no tienen la flexibilidad para mostrar mensajes informativos. Esta pantalla tiene 2 líneas y puede mostrar 16 caracteres en cada línea. No obstante, cuando se conecta con el micrcontrolador, podemos desplazar los mensajes con el software para mostrar información que tiene más de 16 caracteres de longitud.



La pantalla LCD es un dispositivo simple de usar, pero los detalles internos son complejos. La mayoría de las pantallas LCD de 16x2 utilizan un Hitachi HD44780 o un controlador compatible. Sí, hay un microcontrolador dentro de una pantalla de cristal líquido.

Driver l293d



MCP4725 IC es un módulo convertidor digital a analógico de 12 bits que se utiliza para generar voltajes analógicos de salida de (0 a 5V) y se controla mediante la comunicación I2C . También viene con EEPROM de memoria no volátil integrada.

Este IC tiene una resolución de 12 bits. Esto significa que usamos (0 a 4096) como entrada para proporcionar la salida de voltaje con respecto al voltaje de referencia. El voltaje de referencia máximo es de 5V.



Este DAC IC puede interactuar con cualquier microcontrolador utilizando la comunicación I2C. La comunicación I2C requiere solo dos cables SCL y SDA. Por defecto, la dirección I2C para MCP4725 es 0x60 o 0x61 o 0x62. Para mí es 0x61. Usando el bus I2C podemos conectar múltiples MCP4725 DAC IC. Lo único es que necesitamos cambiar la dirección I2C del IC


Luego, ADC se usará para convertir el valor analógico en forma digital. Después de eso, esos valores digitales se envían a MCP4725 a través del bus I2C para convertirlos en señales analógicas utilizando el IC DAC MCP4725.
